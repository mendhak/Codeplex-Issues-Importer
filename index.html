Codeplex-Issues-Importer by mendhak

This script will take work items from your CodePlex project and import them into your GitHub project. It also includes comments.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mendhak/Codeplex-Issues-Importer

get the source code on GitHub : mendhak/Codeplex-Issues-Importer
